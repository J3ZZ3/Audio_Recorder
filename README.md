# Audio Recorder

A simple audio recorder

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/J3ZZ3/Audio_Recorder.git
   cd Audio_Recorder
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   - Create a `.env` file in the root directory and add your environment variables. For example:
     ```
     EXPO_PUBLIC_CLERK_PUBLISHABLE_KEY=your_key_here(obtainable at clerk authentication)
     ```

## Usage

To run the project, use the following command:

  ```bash
  npx expo start
  ```
This will start the Expo development server. You can then open the app on your mobile device using the Expo Go app or the web.

### Screens

- **Login Screen**: Users can log in using their credentials.
- **Register Screen**: New users can create an account.
- **Profile Screen**: Users can view and update their profile information.
- **Audio Recording**: Users can record audio and manage recordings.
